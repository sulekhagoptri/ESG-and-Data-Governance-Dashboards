# ESG-and-Data-Governance-Dashboards
This repository contains two Power BI projects:  1) ESG Country Performance Dashboard  2) Data Governance &amp; Security Dashboard

 1. ESG Country Performance Dashboard
 
 Overview

This dashboard analyzes Environmental, Social, and Governance (ESG) performance across countries using multiple indicator series.
It visualizes long-term trends, regional patterns, and country-level sustainability performance.
